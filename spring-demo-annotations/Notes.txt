BÖLÜM 5 anotasyonlar
00	config xml dosyasına component scan ekle > java classlarına @component anotasyonu ekle > bean'i spring container'dan al.
01	xml içerisine yalnızca component scan ile paket ismi belirttik.
02	java classımıza id'si ile birlikte(thatSillyCoach) @Component anotasyonu ekledik.
03	context.getbean'de o id'yi çağırmamız yeterli oldu.
04	özel bean id eklemeseydik default id TennisCoach.class -> "tennisCoach" olurdu. Yani class'ın camelCase hali.

BÖLÜM 5 dependency injections with annotations and AUTOWIRING
00	CONSTRUCTOR INJECTION 	define dependency interface and class -> create a constructor in your class -> configure DI with @Autowired annotation
01	SETTER INJECTION 	define dependency interface and class -> create setter methods for your injections ->  configure DI with @Autowired annotation
02	FIELD INJECTION 	field belirt "private FortuneService fortuneService" -> configure DI w @Autowired. setter metter gerek yok.
03	öneri : stay consistent on your projects. projende yalnızca bir tanesini seçip kullanmaya dikkat et.
04	aynı interface'ten birden fazla @autowired class olursa (HappyFortuneService, RandomFortuneService, RESTFortuneService etc.) 
	@Qualifier anotasyonunu kullanmalıyız. @Qualifier("id name") (yine özel ad verilmedikçe camelCase yapısı geçerli olabilir.)
